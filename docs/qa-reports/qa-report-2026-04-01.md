- `GET /learn/api/membership-tiers` → 200 PASS

### Login (API)
- `POST /learn/api/users/login` with admin credentials → **200 PASS**
- Token issued successfully. Admin user ID 1 confirmed active.

### Root Cause (Confirmed from Render logs)
PR #59 (i18n foundation) added Payload localization to the `courses`, `articles`, and `pages` collections. This caused Payload to query `courses_locales`, `articles_locales`, and `pages_locales` tables that **do not exist** in the production PostgreSQL database. The migration was not applied.

Render log error (repeated):
```
⨯ Error: Failed query: select "courses"."id" ...
  [cause]: error: relation "courses_locales" does not exist
```

### Action Taken
- Filed GitHub Issue #64: https://github.com/LIMITLESS-LONGEVITY/limitless-paths/issues/64
- Labels: `bug`, `priority:urgent`, `agent:paths`
